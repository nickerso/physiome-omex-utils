# OMEX_manifest_maker

Prerequisites: lxml (pip install)

## list of supported MIME types:
<ul>
<li>.cellml</li>
<li>.sedml  </li>
<li>.py </li>
<li>.csv    </li>
<li>.m      </li>
<li>.xlsx   </li>
<li>.pdf    </li>
<li>.rst    </li>
<li>.txt    </li>
<li>.md    </li>
<li>.png    </li>
<li>.jpeg or .jpg    </li>
<li>.html    </li>
<li>.json    </li>
</ul>
Fork this repository then do a git pull request to add new MIME types or to make changes.
